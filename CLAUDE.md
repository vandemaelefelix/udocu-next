# CLAUDE.md

## Project Overview

udocu — a multilingual website built with Next.js 16, React 19, TypeScript, and Tailwind CSS v4. Content is managed via Prismic CMS. Internationalization is handled by next-intl with `en` and `nl` locales.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript (strict mode)
- **Styling:** Tailwind CSS v4 (PostCSS plugin, `@theme inline` for tokens)
- **CMS:** Prismic (with Slice Machine)
- **i18n:** next-intl — locales: `en`, `nl` (default: `nl`). Routes are prefixed: `/[locale]/...`
- **Fonts:** EB Garamond (Google, placeholder for Adobe Garamond Pro), Helvetica Neue (local, 300/500), Posterman (local). Configured in `src/app/fonts.ts`.
- **Linting:** ESLint (flat config) + Prettier. Enforced via lint-staged + Husky pre-commit hooks.

## Commands

- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run lint` — run ESLint
- `npm run tsc` — type-check (no emit)
- `npm run slicemachine` — start Prismic Slice Machine UI

## Project Structure

```
src/
  app/
    [locale]/          # Locale-scoped routes (layout, page, about, work, contact, blog)
    api/               # API routes (preview, exit-preview, revalidate)
    fonts.ts           # Font definitions
    globals.css        # Tailwind import + theme tokens
    slice-simulator/   # Prismic slice simulator
  assets/
    fonts/             # Local font files (.otf)
    images/            # Static images
  components/          # Shared components (Header, UdocuLogo, etc.)
  i18n/                # next-intl routing and request config
  slices/              # Prismic slice components (auto-generated)
  prismicio.ts         # Prismic client setup
  middleware.ts        # next-intl middleware
```

## Conventions

- Use `@/*` path alias (maps to `./src/*`)
- Tailwind theme tokens are defined in `globals.css` under `@theme inline` always use these instead of hardcoded values
- Font CSS variables: `--font-garamond`, `--font-helvetica-neue`, `--font-posterman`
- Prismic locale mapping: `en` → `en-us`, `nl` → `nl-be` (see `src/prismicio.ts`)
- ESLint ignores `src/slices/**` and `src/slice-machine/**` (auto-generated by Slice Machine)
- Copy should always be added to translation files: en.json, nl.json
